# Thesis
All the codes related to my academic thesis are uploaded here.

The python codes are used to find the mode specifics travel time and travel distance between user specified location to their access or egress metro station and also to find the bus stop near to their locations.

The R codes are used for modelling. Three separate models are developed. One for access, one egress and a combined one merging the access and egress data. The multinomial logit model was used for the three separate models. Mixed logit models are also developed to check if they work better.
